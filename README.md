🛣️ Smart Expressway Simulation System
📌 Overview
This project simulates a highway system using Data Structures and Algorithms. Cities are modeled as nodes and roads as edges in a graph to optimize routes based on distance, time, and toll.
---
🎯 Features
Shortest Path (Dijkstra Algorithm)
Minimum Spanning Tree (Kruskal + DSU)
Fuel Arbitrage Detection (Bellman-Ford)
Vehicle Tracking (Bloom Filter)
Range Queries (Fenwick Tree)

▶️ How to Run
Compile:
g++ src\main.cpp src\DSU.cpp src\CLI.cpp src\Expressway.cpp src\BloomFilter.cpp src\Fenwick.cpp -o expressway
Run:
.\expressway.exe
